## 🔬 C. Files (Curiosity Files)

In this repository, you’ll find a series of C++ files that start with the letter **`C`** instead of a number.

- `C` stands for **Curiosity**  
- 🧪 These files contain experiments and exploratory code  
- 📝 They are not part of the main structured sequence (0, 1, 2, …)  
- 💡 They include extra examples, variations, and interesting small programs  

Think of them as a research notebook — a place for testing ideas and exploring concepts beyond the main lessons.

---

### 📚 Recommended Study Order

1. ✅ Start with the **main numbered files**:  
   `0`, `1`, `2`, `3`, …  

2. 🔎 After completing a topic, explore the related **`C.` files** to:
   - Reinforce understanding  
   - See alternative implementations  
   - Explore creative or fun variations  

These files are meant to deepen understanding and encourage curiosity-driven learning.
